# Project Plan — goclaw-setup-my-pc

**Качество важнее скорости.**  
Мы не переходим к следующему этапу, пока текущий не доведён до хорошего уровня.

---

## Принципы качества

1. Каждый skill должен быть достаточно подробным, чтобы агент мог реально им пользоваться.
2. Опасные действия всегда требуют подтверждения.
3. Приоритет локальной документации над интернетом.
4. Официальная документация используется как источник правды (особенно OpenWrt).
5. Не копируем skills слепо — адаптируем под наши требования (безопасность, USB, hybrid).
