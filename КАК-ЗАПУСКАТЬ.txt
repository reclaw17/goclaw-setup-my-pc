КАК ЗАПУСКАТЬ
=============

Вариант A (как Cachy Update)
----------------------------
1. Обнови файлы проекта с GitHub
2. Правый клик по файлу 1-подготовить.desktop → Allow Launching / Разрешить запуск
3. Двойной клик по:
   - 1-подготовить.desktop
   - 2-старт.desktop
   - 3-обновить.desktop

Откроется окно терминала, будет видно, что происходит.
В конце: Нажмите Enter для выхода.

Вариант B (самый надёжный)
--------------------------
1. В папке проекта: правый клик → Открыть терминал
2. Команды:
   bash подготовить.sh
   bash старт.sh
   bash обновить.sh

Почему раньше "ничего не было"?
------------------------------
Обычный двойной клик по .sh часто запускает скрипт без окна.
Теперь скрипт сам старается открыть Konsole.
