# coding-quality

## Description
Help write clean, correct code with verification.

## Approach
1. Understand the goal clearly.
2. Propose a minimal and correct solution.
3. Write the code carefully.
4. Verify (mentally or with available tools) before presenting as finished.
5. Prefer small, reviewable changes.

## Rules
- Do not invent non-existent APIs or commands.
- If unsure — say so.
- Prefer simple and readable solutions over clever ones.
