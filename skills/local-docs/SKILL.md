# local-docs

## Description
Search and use the local offline documentation base on the USB drive.

## When to use
- Any question about OpenWrt, CachyOS, Arch Linux, Windows 11, AdGuard, Amnezia, networking, VPN, etc.
- Prefer this skill **before** using internet search.

## How it works
1. Look in the `docs/` folder on the USB.
2. Find relevant Markdown files.
3. Answer based on the found documentation.
4. If nothing useful is found — say so and offer to search online.

## Rules
- Always try local docs first.
- Do not invent information that is not in the docs.
- User can add new documents to `docs/custom/`.
