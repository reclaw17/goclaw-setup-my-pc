# Bootstrap files for GoClaw Lite

Upstream GoClaw injects workspace files such as `AGENTS.md`, `SOUL.md`, `TOOLS.md` into the agent context (see official docs/07).

## How to use

1. Start GoClaw Lite (AppImage).  
2. Open the agent / workspace settings.  
3. Copy the contents of these files into the corresponding bootstrap slots **or** place them in the workspace folder Lite uses for bootstrap (see upstream docs).  

Do not put API keys here.
