# goclaw-setup-my-pc

Портативный AI-агент для настройки компьютера и OpenWrt-роутера.

Работает с флешки.  
Подходит даже если у тебя **нет опыта в программировании**.

---

## Как запустить

### 1. Скопируй проект на флешку

### 2. Подготовка (один раз)

**Windows:** `подготовить.bat`  
**Linux:** `bash подготовить.sh`

### 3. Ключи (для online-режима)

Открой `.env` и вставь API-ключи.

### 4. Запуск

**Windows:** `старт.bat`  
**Linux:** `bash старт.sh`

### 5. Обновление

**Windows:** `обновить.bat`  
**Linux:** `bash обновить.sh`

Скрипт спросит, что обновлять:
- файлы проекта
- программы
- всё вместе

Без твоего «да» ничего менять не будет.

---

## Что умеет

- CachyOS / Arch Linux и Windows 11
- OpenWrt (сначала бэкап)
- Работа без интернета
- Подтверждение перед опасными действиями
- Русский и английский

---

## Важно

1. Файл `.env` никому не отправляй
2. Папки `models/` и `docs/custom/` при ручном обновлении не затирай
3. Локальная модель сама не обновляется — только по желанию
4. Перед настройкой роутера агент должен предложить бэкап

---

## Если что-то не работает

1. Сначала `подготовить`
2. Потом `старт`
3. Запускай файлы из папки проекта на флешке
